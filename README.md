The prot2vec is a protein pre-trained dictionary. In our work, we converted protein sequences into real-value matrix. 
The smi2vec is a drug dictionary which can be used to convert drug string into matrix. The word2vec_30.model converted protein sequence into vector. 
main_sent.py is a  main file for runing our SENT-DTI method. 
The interaction_feature_0.csv is a embedding learned by pre-trained birectional encoder-decoder.
